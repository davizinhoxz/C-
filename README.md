## 📘 Projeto em C

Este repositório foi criado no âmbito da disciplina de programação e tem como objetivo desenvolver e praticar a linguagem C.

A linguagem C é uma das bases da programação moderna, sendo muito utilizada para compreender conceitos fundamentais como variáveis, estruturas de controlo, funções e manipulação de memória.

### 🎯 Objetivos do projeto
- Aprender os conceitos básicos da programação em C  
- Desenvolver o raciocínio lógico  
- Resolver problemas através de algoritmos  
- Criar pequenos programas funcionais  

### 🧠 Conteúdos abordados
- Estruturas condicionais (if, else)  
- Ciclos (for, while)  
- Funções  
- Arrays e strings  

### 💻 Sobre o repositório
Este repositório contém exercícios e exemplos práticos realizados durante as aulas, servindo como apoio ao estudo e à evolução na programação.  

Os projetos aqui presentes podem ser utilizados para aprendizagem e melhoria das competências em programação.
